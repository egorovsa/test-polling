# monopolling

QA-обход собственного сайта на [Playwright](https://playwright.dev/): имитирует
поведение посетителя (заход → пауза → плавный скролл → клик по случайной
внутренней ссылке) для функциональной/нагрузочной проверки.

> **Только свой сайт.** Весь трафик помечен как тестовый, чтобы его можно было
> отфильтровать в аналитике и не засчитать как реальных посетителей.

## Тестовая пометка

Каждый запрос помечается **на уровне всех HTTP-запросов** двумя способами:

- `User-Agent: Mozilla/5.0 (test) qa-playwright-test`
- заголовок `X-Test-Traffic: qa-playwright-test`

В аналитике/логах отфильтруйте трафик по этому маркеру, чтобы он не попадал в
метрики посещаемости. Менять маркер — переменной `TEST_MARKER` в `index.js`.

## Установка

```bash
npm install            # поставит playwright и скачает chromium (postinstall)
```

Если браузер не скачался автоматически:

```bash
npx playwright install chromium
```

## Запуск

```bash
BASE_URL="https://example.com" VISITS=100 npm start
```

| Переменная | По умолчанию             | Назначение                  |
| ---------- | ------------------------ | --------------------------- |
| `BASE_URL` | `https://твой-сайт.example` | Адрес тестируемого сайта |
| `VISITS`   | `100`                    | Число сессий                |

По умолчанию `BASE_URL` — заглушка, замените на свой адрес.

## Что делает один «визит»

1. Открывает `BASE_URL`, ждёт `networkidle`.
2. Пауза 1–3 c («читает» страницу).
3. Плавный скролл до низа страницы.
4. Клик по случайной внутренней ссылке, пауза 1.5–4 c.
5. Закрывает контекст; пауза 2–8 c перед следующей сессией.
